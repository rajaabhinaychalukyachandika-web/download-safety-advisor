# Download Safety Advisor

Download Safety Advisor is a web-based project that helps users understand whether a file or link is safe to download or not.

This project is designed to create awareness about unsafe downloads and basic online safety.

## Features
- Simple and user-friendly interface
- Helps users check download safety
- Web-based application

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Purpose
This project is created as a learning project to improve my frontend and web development skills.
